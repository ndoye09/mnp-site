# MBAO Notre Priorité - Site Web

## 📋 Description

Site web moderne et professionnel pour le mouvement communautaire **MBAO Notre Priorité**. Ce site présente les activités, services et la boutique officielle du mouvement.

## 🗂️ Structure du Site

### Pages Principales

1. **index.html** - Page d'accueil
   - Section héros avec appel à l'action
   - Présentation du mouvement
   - Domaines d'action (6 sections)
   - Galerie de photos intégrée
   - Bureau exécutif
   - Boutique officielle
   - Sections et commissions

2. **devenir-membre.html** - Page d'adhésion
   - Formulaire d'inscription complet
   - 5 sections: Infos personnelles, Coordonnées, Affiliation, Motivations, Conditions

3. **galerie.html** - Galerie complète
   - Grid d'images responsif
   - Hover effects élégants
   - Lien vers page dédiée

4. **boutique.html** - Boutique officielle
   - 3 produits: Pull, T-shirt, Casquette
   - Pricing en FCFA
   - Commande via WhatsApp
   - Modes de paiement: Wave, Orange Money
